📌 Panoramica

Il modulo Esportazione GEDCOM della v3 e' un ambiente di analisi genealogica assistita: prende testi, immagini o documenti selezionati, usa un provider IA configurato in ATK-Pro e genera un file .ged nella cartella di output scelta.

Si apre dal menu Servizi → Esportazione GEDCOM. Il nome pubblico resta legato al formato GEDCOM, ma il lavoro effettivo comprende anche estrazione genealogica, uso di note paleografiche, eventuale incremento di una base esistente e gestione delle chiavi IA.

Il risultato deve essere verificato prima di importarlo in un software genealogico: l'IA puo' aiutare a strutturare dati e relazioni, ma non sostituisce la lettura critica delle fonti.

🖥️ Interfaccia

1. Metodo e input universale

La prima sezione permette di scegliere il metodo di lavoro e i file da analizzare.

  • Auto-Discovery: scelta adatta quando vuoi lasciare al modulo il compito di interpretare il materiale selezionato.
  • Vision IA: usa un modello capace di leggere immagini quando il materiale di partenza e' visivo.
  • Refinery LLM: utile per lavorare su testo gia' estratto o trascritto.
  • Input: seleziona uno o piu' atti, immagini, trascrizioni o documenti compatibili con il flusso scelto.

2. Incrementa base esistente

Il pulsante CARICA BASE consente di indicare una base genealogica gia' esistente in formato GEDCOM o CSV. Se non carichi nulla, il modulo crea un nuovo lavoro.

3. Banca dati note paleografiche

Questa sezione collega il lavoro genealogico alle tipologie documentali e alle note operative. Puoi scegliere il tipo di atto, richiamare una nota salvata, crearne una nuova, modificarla o eliminarla.

Le note servono a spiegare al modello abbreviazioni, formule locali, lingua del documento, regole di parentela, dubbi di lettura o criteri di estrazione.

4. Destinazione output

Scegli la cartella in cui salvare il risultato. Il file generato usa un nome del tipo genealogia_*.ged.

5. Motore IA e caveau chiavi

  • Provider: scegli il servizio IA tra quelli configurati, ad esempio Claude, OpenAI, Gemini, Mistral, xAI, DeepSeek, Groq, HuggingFace o Ollama.
  • Override modello: campo opzionale per indicare un modello specifico supportato dal provider scelto.
  • GESTISCI CAVEAU CHIAVI (CSV): apre la gestione delle chiavi disponibili.
  • AVVIA ESTRAZIONE COMPLETA: avvia l'analisi e la generazione del GEDCOM.

📝 Flusso consigliato

  1. Prepara input leggibili: trascrizioni, immagini o file gia' organizzati per atto.
  2. Scegli il metodo di analisi piu' adatto al materiale.
  3. Se stai proseguendo un lavoro esistente, carica la base GEDCOM o CSV.
  4. Seleziona la tipologia di atto e aggiungi una nota paleografica concreta.
  5. Scegli cartella di output, provider e chiave.
  6. Avvia l'estrazione completa.
  7. Controlla il file .ged generato prima di importarlo in altri programmi.

⚠️ Avvertenze

  • Il GEDCOM prodotto e' una base di lavoro, non una prova definitiva.
  • Relazioni familiari, date, luoghi e nomi devono essere verificati sulle fonti originali.
  • Gli errori di OCR, trascrizione o interpretazione possono propagarsi nel file genealogico.
  • Prima di importare il GEDCOM in software esterni, conserva una copia del file originale e controlla eventuali duplicati.
  • Non inviare a provider esterni documenti riservati o soggetti a restrizioni senza aver verificato condizioni d'uso, privacy e autorizzazioni.

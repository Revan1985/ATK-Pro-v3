# ATK-Pro – Воссстановитель плитки предков
**Примечание:** Этот проект разрабатывается, поддерживается и обеспечивается поддержкой полностью одним человеком. Любые отзывы, сообщения или вклад приветствуются, но за разработкой не стоит команда или корпоративная структура.
## Описание
ATK-Pro — это продвинутый инструмент для восстановления, архивирования и просмотра оцифрованных изображений и генеалогических документов с портала Antenati. Проект поддерживает многоязычное управление и распространение в виде автономного приложения для Windows.
## Основные функции
- Автоматическая реконструкция изображений из тайлов IIIF
- Многоязычная поддержка (20 языков)
- Современный графический интерфейс (Qt)
- Создание автономного EXE и многоязычного установщика
## Установка
1. Скачайте установщик ATK-Pro-Setup-v2.0.exe или автономный исполняемый файл ATK-Pro.exe из раздела релизов.
1. Следуйте инструкциям на экране, чтобы завершить установку.
1. Запустите ATK-Pro из меню «Пуск» или из папки установки.
## Структура проекта
- `src/` – Основной исходный код (GUI, логика, модули)
- `assets/` – Многоязычные ресурсы (руководства, шаблоны, материалы)
- `locales/` – Файлы перевода .ini для каждого языка
- `docs_generali/` – Глоссарии, общая документация, дорожная карта
- `scripts/` – Скрипты для обслуживания, перевода, валидации
- `tests/` – Автоматическое тестирование и тестирование покрытия
- `dist/` – Вывод сборки/установщика

# CHANGELOG – ATK-Pro

## v2.2.0 (22 de março de 2026)
- **Novo**: PDF como formato de saída independente — selecionável sozinho ou combinado com PNG/JPG/TIFF
- O PDF é gerado automaticamente a partir de imagens IIIF reconstruídas, sem ferramentas externas
- Caixa de seleção PDF adicionada ao Passo 1 do guia; seção PDF atualizada nos 20 idiomas
- Número de versão removido do rodapé de todos os arquivos de texto
- Capturas de tela obsoletas `window3.webp` removidas de todos os assets de idioma
- Correções: arquivos temporários `_pdftmp.png`, ordem das páginas no PDF, marcadores de posição localizados

## v2.0 (2026)
- Lançamento público versão 2.0: reconstrução de imagens a partir de tiles IIIF, suporte multilíngua, interface Qt, build EXE e instalador
- Otimização em lote, gerenciamento de erros, log avançado, automação completa
- Atualização e racionalização da documentação
## v1.x (2023-2025)
- Versões experimentais e protótipos
- Melhorias progressivas em fluxo de trabalho, compatibilidade, exportação de dados

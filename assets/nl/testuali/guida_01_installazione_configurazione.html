1. INSTALLAZIONE E REQUISITI

────────────────────────────────────────────────────────────────────────────────

SISTEMA OPERATIVO

  • Windows: Windows 10 o successivo (64-bit) — Professional, Home, Server
  • Linux: Ubuntu 20.04+ o distribuzioni compatibili (pacchetto .deb o archivio .tar.gz)
  • macOS: macOS 11 Big Sur o successivo — distribuzione .dmg (test su hardware reale in corso)
  • 64-bit obbligatorio su tutte le piattaforme
  • Almeno 2 GB di RAM disponibili (4 GB consigliati per elaborazione batch)
  • 1 GB di spazio libero su disco

INSTALLAZIONE

🪟 Windows

  1. Scarica ATK-Pro-Setup-vx.x.x.exe dalla pagina Release ufficiale
  2. Fai doppio click sul file installer
  3. Seleziona "Esegui con privilegi di amministratore", se richiesto
  4. Segui la procedura guidata (Next → Next → Install)
  5. Durante l'installazione verrà richiesta la selezione della lingua (20 disponibili)
  6. Al termine, spunta "Avvia ATK-Pro" per avviare il programma
  7. Accetta eventuali avvisi di Windows Firewall/Defender

🐧 Linux

  1. Scarica ATK-Pro-vx.x.x.deb (Debian/Ubuntu) o .tar.gz dalla pagina Release
  2. Pacchetto .deb: sudo dpkg -i ATK-Pro-vx.x.x.deb oppure doppio click in Nautilus/Files
  3. Archivio .tar.gz: estrai e avvia ./ATK-Pro dalla cartella estratta
  4. La lingua viene selezionata al primo avvio e salvata nel file di configurazione utente

🍎 macOS

  1. Scarica ATK-Pro-vx.x.x.dmg dalla pagina Release
  2. Apri il file .dmg e trascina ATK-Pro nella cartella Applicazioni
  3. Al primo avvio: clic destro → Apri (necessario per applicazioni non notarizzate)
  4. La lingua viene selezionata al primo avvio e salvata nel file di configurazione utente

POSIZIONI DI DEFAULT

🪟 Windows

  • Programma: C:\Program Files\ATK-Pro\
  • Output utente: C:\Users\[TuoNome]\Documents\ATK-Pro\output\
  • Input (opzionale): C:\Users\[TuoNome]\Documents\ATK-Pro\input\

🐧 Linux

  • Programma: /opt/ATK-Pro/ (deb) oppure cartella estratta (.tar.gz)
  • Output utente: ~/Documents/ATK-Pro/output/
  • Input (opzionale): ~/Documents/ATK-Pro/input/

🍎 macOS

  • Programma: /Applications/ATK-Pro.app
  • Output utente: ~/Documents/ATK-Pro/output/
  • Input (opzionale): ~/Documents/ATK-Pro/input/

REQUISITI DI RETE

  • Connessione Internet stabile (obbligatoria per accedere a IIIF Manifests)
  • Firewall: Consenti connessioni HTTPS in uscita (porta 443)
  • Proxy aziendale: Se in ambiente corporate, configura le credenziali proxy nel menu

📂 CONFIGURAZIONE CARTELLE

Alla prima esecuzione, il programma crea automaticamente le seguenti cartelle:

  • Windows: C:\Users\[TuoNome]\Documents\ATK-Pro\output\
  • Windows: C:\Users\[TuoNome]\Documents\ATK-Pro\output\doc\ (default documenti)
  • Windows: C:\Users\[TuoNome]\Documents\ATK-Pro\output\reg\ (default registri)
  • Linux/macOS: analoghe sotto ~/Documents/ATK-Pro/output/
  • Cartella input\ analoga (opzionale, per input file)

Durante l'elaborazione vengono create cartelle temporanee per i tile scaricati (tiles_canvas_X/) e, se richiesto il PDF, file temporanei per la sua generazione.

Tutte le cartelle e i file temporanei vengono eliminati automaticamente al termine dell'elaborazione.

Rimangono solo i file finali elaborati (immagini nei formati scelti, PDF, manifest e metadati).

Per modificare le cartelle di output usa Menu → Impostazioni → "Seleziona cartelle output". Le scelte vengono memorizzate e ripristinate automaticamente alla sessione successiva.

3. INTERFACCIA PRINCIPALE

────────────────────────────────────────────────────────────────────────────────

LAYOUT DELLA FINESTRA

La finestra principale di Antenati Toolkit Pro ha una struttura semplice e intuitiva.

📝 LEGENDA TESTATA:

  • [🔲] Icona del programma (Antenati Toolkit Pro logo)
  • "ATK-Pro" Titolo della finestra
  • [_] Riduci a icona (minimizza la finestra)
  • [□] Espandi/Collassa (massimizza o ripristina la finestra)
  • [✕] Chiudi (termina il programma con banner di supporto)

In calce è riportato il motto latino “Qui ante nos venerunt, in memoria nostra vivunt”, a ricordare l’importanza delle radici e della memoria collettiva.

BARRA DEI MENU

Nella parte superiore della finestra, sotto la testata:

[File input] [Output] [Servizi] [Documenti] [Impostazioni]

Ogni menu contiene pulsanti e opzioni specifiche (vedi sezione successiva).

MENU DISPONIBILI

1. FILE INPUT

Gestisce il caricamento dei record IIIF:

├─ Esempio input
│  └─ Visualizza il file di esempio (mostra formato coppie: modalità D/R + URL)
├─ Apri input
│  └─ Seleziona un file .txt con i tuoi record IIIF
│     (Deve contenere coppie: modalità D/R su riga 1, URL su riga 2)
└─ Modifica input
   └─ Modifica il file input attualmente caricato
      (Aggiungi/rimuovi record, cambia URL, salva modifiche)

2. OUTPUT

Gestisce elaborazione e output:

├─ Elabora
│  └─ Avvia il download, la ricostruzione e il salvataggio delle immagini
│     (Mostra finestra di progresso in tempo reale)
├─ Apri cartella output
│  └─ Apre in Esplora Risorse la cartella con i file elaborati
└─ Visualizza elaborazioni precedenti
   └─ Lista dei record già elaborati

3. SERVIZI

Elenco dei servizi integrati:

├─ Visualizzazione Immagini
│  └─ Viewer immagini scaricate e/o PDF generati
├─ Visualizzazione Metadati JSON
│  └─ JSON originali e metadati (genealogici, tecnici, EXIF)
├─ OCR Avanzato
│  └─ Testi manoscritti ottocenteschi, latino ecclesiastico, tardo-medievali
├─ Traduzione OCR
│  └─ Traduzione automatica testi estratti nelle lingue selezionate
└─ Esportazione GEDCOM
   └─ GEDCOM, Gramps, Family Tree Maker, Ancestry, MyHeritage, WikiTree, etc.

4. DOCUMENTI

Accesso a risorse e documentazione:

├─ Disclaimer - Clausole legali e termini d'uso
├─ Presentazione autore - Background dell'autore del progetto
├─ Presentazione progetto - Storia, motivazioni, roadmap di ATK-Pro
├─ Guida - Questo documento (guida operativa completa)
└─ Informazioni - Versione, copyright, email di contatto

5. IMPOSTAZIONI

Configurazione generale del programma:

├─ Cambia lingua
│  └─ Seleziona la lingua dell'interfaccia (richiede riavvio)
├─ Seleziona formati immagine
│  └─ Scegli tra PNG, JPG, TIFF e PDF (almeno 1 obbligatorio)
│     PDF può essere selezionato da solo o insieme ai formati immagine
│     La scelta viene memorizzata e pre-selezionata automaticamente alla sessione successiva
├─ Seleziona cartelle output
│  └─ Apre prima la scelta della modalità cartelle, poi le finestre di selezione:
│     • Cartella unica per tutti    → una sola cartella per documenti e registri
│     • Cartelle separate doc/reg   → una cartella per tutti i doc, una per tutti i reg
│     • Cartella per ogni record    → una cartella distinta per ciascun record
│     La modalità e le cartelle vengono memorizzate e ripristinate alla sessione successiva
├─ Esporta configurazione
│  └─ Salva le impostazioni attuali in un file
├─ Importa configurazione
│  └─ Carica un file di impostazioni precedentemente salvato (ripristina tutte le preferenze e percorsi)
├─ Verifica aggiornamenti
│  └─ Controlla la disponibilità di nuove versioni del programma (funzione in sviluppo)
└─ Chiudi
    └─ Termina il programma (mostra banner di supporto PayPal.me)

📌 Panoramica

Il modulo Esportazione GEDCOM è in fase di pianificazione. Permetterà di esportare le relazioni familiari estratte dai documenti genealogici elaborati da ATK-Pro nel formato GEDCOM (GEnealogical Data COMmunication), lo standard internazionale per l'interscambio di dati genealogici.

Una volta sviluppato, il modulo si occuperà di aggregare le entità estratte dall'OCR (nomi, date, relazioni di parentela) e di strutturarle in un file GEDCOM compatibile con i principali software genealogici.

🌳️ Interoperabilità Pianificata

Software Genealogici Compatibili (Obiettivo)

L'obiettivo è garantire la compatibilità con i principali software e piattaforme genealogici:

  • Ancestry
  • Family Historian
  • Family Tree Maker
  • Findmypast
  • FamilySearch
  • Geni
  • Geneanet
  • Gramps (open source)
  • Legacy Family Tree
  • MyHeritage
  • Reunion (macOS)
  • RootsMagic
  • WikiTree

📌 Nota: L'elencazione è puramente indicativa e non vincolante allo sviluppo dell'interoperabilità con ciascuna piattaforma o software.

⚙️ Funzionalità Previste

⚠️ Nota: Le funzionalità descritte di seguito rappresentano una bozza ipotetica delle possibili integrazioni future. Non vi è ancora alcun riscontro di fattibilità tecnica né alcun impegno di sviluppo. Le specifiche potranno variare o essere abbandonate in qualsiasi momento.

  • Estrazione automatica di entità (nomi, date, relazioni) dai testi OCR tramite LLM
  • Linkage ed entity extraction tra registri multipli analizzati in blocco
  • Generazione di file .ged conformi allo standard GEDCOM 5.5.1 e GEDCOM 7.0
  • Verifica e deduplicazione automatica degli individui identificati
  • Esportazione con metadati delle fonti (riferimento al documento originale, pagina, data)

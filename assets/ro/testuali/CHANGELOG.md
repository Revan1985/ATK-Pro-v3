# Jurnal de modificări – ATK-Pro

## v2.2.0 (22 martie 2026)
- **Nou**: PDF ca format de ieșire independent — selectabil singur sau combinat cu PNG/JPG/TIFF
- PDF-ul este generat automat din imaginile IIIF reconstruite, fără instrumente externe
- Casetă de selectare PDF adăugată la Pasul 1 al ghidului; secțiunea PDF actualizată în toate cele 20 de limbi
- Numărul de versiune eliminat din subsolul tuturor fișierelor text
- Capturi de ecran `window3.webp` învechite eliminate din toate resursele de limbă
- Remedieri: fișiere temporare `_pdftmp.png`, ordinea paginilor în PDF, marcatori de poziție localizați

- Lansare publică versiunea 2.0: reconstrucție imagini din tile IIIF, suport multilingv, interfață Qt, build EXE și installer
- Optimizare batch, gestionare erori, logging avansat, automatizare completă
- Actualizare și raționalizare documentație
## v1.x (2023-2025)
- Versiuni experimentale și prototipuri
- Îmbunătățiri progresive ale fluxurilor de lucru, compatibilității, exportului de date

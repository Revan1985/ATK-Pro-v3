📌 Panoramica

Il modulo OCR Avanzato (Trascrizione Diplomatica) estrae il testo da immagini e documenti PDF tramite modelli di intelligenza artificiale multimodali (LLM Vision). È progettato specificamente per documenti storici: manoscritti, atti notarili, registri parrocchiali, lettere d'archivio in italiano ottocentesco, latino o dialetto.

A differenza dei motori OCR tradizionali (Tesseract, ecc.), il modulo interpreta la grafia nel contesto del documento, riconosce abbreviazioni e formule ricorrenti, e può mantenere la struttura del layout originale in formato TEI XML.

Si apre dal menu Strumenti → OCR Avanzato dell'applicazione principale.

🖥️ Interfaccia — Guida Passo per Passo

📂 Passo 1 — Selezione dei File

Clicca “Seleziona File (Immagini/PDF)” per aprire il selettore. Puoi selezionare più file contemporaneamente tenendo premuto Ctrl o Shift.

Formati accettati:

  • Immagini: .jpg, .jpeg, .png, .tiff, .tif
  • PDF: qualsiasi documento PDF (scansionato o nativo)

Il numero di file selezionati viene mostrato accanto all'etichetta. Ogni file viene elaborato separatamente e genera i propri file di output.

🤖 Passo 2 — Selezione del Provider IA e API Key

Seleziona il provider nel menu a tendina “Seleziona Provider IA”:

Inserisci la tua API Key nel campo “API Key”. La chiave è mascherata per sicurezza e viene salvata automaticamente nelle preferenze. La stessa chiave viene condivisa con il Modulo Traduzione.

📄 Passo 3 — Formati di Output

Seleziona uno o più formati di output (tutti e tre selezionati per default):

  • Testo Semplice (.txt): Trascrizione piana, massima compatibilità. Ideale per elaborazioni successive (traduzione, indicizzazione).
  • Documento Word (.docx): Testo formattato, editabile con Microsoft Word o LibreOffice.
  • XML TEI (.xml): Formato standard accademico per edizioni digitali critiche (Text Encoding Initiative). Mantiene struttura, layout e metadati del documento.

I file vengono salvati nella cartella di destinazione che sceglierai al momento dell'avvio OCR.

📝 Passo 4 — Istruzioni Aggiuntive (Opzionale ma Consigliato)

Il campo “Istruzioni Aggiuntive per l'IA” permette di personalizzare il comportamento del modello per ogni sessione di lavoro:

  • Indicare il dialetto o la variante linguistica del testo
  • Segnalare abbreviazioni ricorrenti e il loro significato
  • Specificare se ignorare timbri, annotazioni marginali o francobolli
  • Richiedere la conservazione di formule specifiche

Esempi pratici:

  • “I nomi dei mesi sono in dialetto siciliano ottocentesco. Non correggere.”
  • “Ignora i timbri a margine e le annotazioni in matita rossa.”
  • “Le abbreviazioni 'fu' e 'q.' indicano 'figlio di' e 'quondam' (defunto).”

Archivio istruzioni: Le istruzioni usate più frequentemente possono essere salvate cliccando “Salva” e richiamate dal menu a tendina “Istruzioni Salvate” nelle sessioni successive.

🎓 Passo 5 — Trascrizione di Riferimento (Few-Shot, Opzionale)

Il campo “Trascrizione di Riferimento” implementa la tecnica del few-shot learning: fornendo all'IA un esempio di trascrizione corretta, essa imparerà a riconoscere la calligrafia specifica del documento.

Come usarla:

  1. Trascrivi manualmente e con precisione la prima pagina (o le prime righe) del registro
  2. Incolla il testo trascritto nel campo “Trascrizione di Riferimento”
  3. Avvia l'OCR: il modello userà quella trascrizione come guida per tutto il resto del documento

Risultato: Incremento significativo della precisione, soprattutto su grafie corsive difficili, abbreviazioni insolite o dialettismi locali.
Il campo “Cancella Testo” (✖) svuota la trascrizione e ripristina la modalità standard.

🔍 Passo 6 — Revisione Interattiva (Opzionale)

Spuntare “Abilita Finestra di Revisione Interattiva” attiva una modalità interattiva che si apre dopo ogni elaborazione:

  • Pannello sinistro: immagine originale con zoom (rotella del mouse) e pan
  • Pannello destro: testo OCR estratto, modificabile liberamente
  • Pulsante “Approva e Genera File”: salva il testo revisionato nei formati scelti
  • Pulsante “Salta Pagina”: scarta l'elaborazione corrente e passa alla pagina successiva

Utile per verificare in tempo reale la qualità dell'OCR e correggere errori prima del salvataggio definitivo. Per batch di grandi dimensioni è preferibile disattivare la revisione interattiva e correggere il testo a posteriori.

▶️ Passo 7 — Avvio OCR

Clicca “Avvia OCR”. Verrà richiesta la cartella di destinazione per i file di output.

La barra di avanzamento mostra il progresso (file corrente / totale file). L'elaborazione avviene in un thread separato: l'interfaccia rimane responsiva durante il processing.

Al termine, una finestra di dialogo confermerà il completamento o segnalerà eventuali errori.

🔑 Come Ottenere le API Key

✅ Google Gemini — Gratuito (Consigliato)

  1. Vai su https://aistudio.google.com/
  2. Accedi con un Account Google (Gmail)
  3. In alto a sinistra, clicca “Get API Key” o “Crea API key”
  4. Seleziona o crea un progetto Google Cloud (puoi usare quello di default)
  5. Clicca “Crea API key nel progetto esistente”
  6. Copia la chiave generata e incollala nel campo API Key di ATK-Pro

Limiti gratuiti (inizio 2026): Supporto per la famiglia Gemini 2.5 (Flash e Pro). Google ha introdotto quote dinamiche: i tuoi limiti (RPM/TPM/RPD) sono ora consultabili direttamente nel dashboard di AI Studio. Indicativamente sono disponibili circa 15 richieste/minuto, ideali per l'uso amatoriale e l'elaborazione di singoli registri.

💵 OpenAI (GPT-4o)

  1. Vai su https://platform.openai.com/
  2. Crea un account o effettua l'accesso
  3. Nel menu laterale sinistro, seleziona “API keys”
  4. Clicca “Create new secret key”, assegna un nome descrittivo
  5. Copia immediatamente la chiave: non verrà mostrata di nuovo
  6. Carica crediti tramite Billing → Add payment method

Nota: servizio a pagamento (pay-per-use). OCR con GPT-4o costa indicativamente $0.01–$0.03 per immagine ad alta risoluzione.

💵 Anthropic (Claude 3.5)

  1. Vai su https://console.anthropic.com/
  2. Crea un account o effettua l'accesso
  3. Nella dashboard, seleziona “API Keys”
  4. Clicca “Create Key”, assegna un nome
  5. Copia la chiave generata
  6. Carica crediti dalla sezione Billing

Nota: servizio a pagamento. Claude 3.5 eccelle in testi lunghi e strutturati, ma può essere meno preciso di GPT-4o su grafie molto difficili.

💡 Consigli per Risultati Ottimali

  • Qualità immagini: Immagini nitide a 300+ DPI producono risultati nettamente migliori. Evita JPEG con compressione elevata (artefatti).
  • Usa sempre le Istruzioni Aggiuntive: Anche poche righe di contesto aumentano significativamente la precisione per documenti dialettali o con grafia insolita.
  • Prima pagina come riferimento: Trascrivi manualmente la prima pagina e usa il campo “Trascrizione di Riferimento”; il miglioramento sulle pagine successive è spesso notevole.
  • Batch di test: Per grandi batch (100+ immagini), esegui prima un test su 5-10 file con Revisione Interattiva attiva prima di procedere sul lotto completo.
  • TEI XML: Seleziona il formato XML TEI se prevedi di depositare la trascrizione in archivi digitali accademici o di condividerla con altri ricercatori.
  • Salva le istruzioni: Ogni serie archivistica ha caratteristiche proprie. Salva un set di istruzioni specifico per ogni registro o fondo trattato.

📚 Modulo 04: Visualizzazione Metadati JSON

⬅️ Mod.03 — Visualizzazione Immagini 🏠 Indice Principale Mod.05 — Funzioni Avanzate OCR ➡️

📌 Panoramica

Il Viewer Metadati JSON di ATK-Pro è dedicato alla visualizzazione dei file JSON generati durante l'elaborazione dei documenti. Si apre dal menu Strumenti dell'applicazione.

Il viewer apre qualsiasi file .json. Cerca automaticamente un'immagine con lo stesso nome nella stessa cartella (estensioni: .jpg, .jpeg, .png, .tif, .tiff) e, se la trova, la mostra nel pannello destro.

🌳️ Interfaccia

La finestra è divisa in due pannelli affiancati:

Pannello Sinistro — Tabella Metadati

I dati del file JSON vengono visualizzati come una tabella a due colonne:

  • Campo: il nome della chiave JSON
  • Valore metadato: il valore corrispondente

Vengono visualizzati solo i campi di primo livello del JSON. I campi il cui nome inizia con _ (underscore) sono omessi perché riservati all'uso interno. Strutture annidate (oggetti o array) appaiono come rappresentazione testuale nella colonna Valore.

Pannello Destro — Anteprima Immagine

Se viene trovata un'immagine abbinata, questa viene caricata con le stesse funzionalità del Viewer Immagini (zoom con rotella, pulsanti +/−, Adatta alla finestra, 1:1, pan con trascinamento).

Se non viene trovata alcuna immagine abbinata, il pannello mostra “Nessuna immagine abbinata trovata”.

Toggle Pannelli

  • Pulsante “◄ Nascondi metadati”: Nasconde il pannello sinistro con i metadati
  • Pulsante “► Mostra metadati”: Ripristina il pannello metadati
